# 学習メモ置き場

このフォルダには、自分の学習メモを自由に追加できます。

## 📝 メモの書き方

### ファイル名の例
```
2025-11-30-di-learning.md      # 日付 + トピック
mvc-tips.md                     # トピックのみ
questions.md                    # 疑問点をまとめる
```

### テンプレート

```markdown
# 学習メモ: [トピック名]

日付: YYYY/MM/DD

## 学んだこと

- ポイント1
- ポイント2

## わからなかったこと

- 疑問点1
- 疑問点2

## 次にやること

- [ ] タスク1
- [ ] タスク2

## 参考リンク

- [リンク名](URL)
```

---

## 💡 メモのコツ

1. **こまめに書く**: 忘れる前に記録
2. **自分の言葉で**: コピペではなく理解したことを書く
3. **疑問も残す**: 後で調べるきっかけになる
4. **コード例を入れる**: 理解を深める
